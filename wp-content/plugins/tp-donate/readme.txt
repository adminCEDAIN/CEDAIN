TP Donation - A complete donation plugin for WordPress.

== Description ==

A complete hotel booking plugin for WordPress.

**Features**

* Complete Donation System
* Campaign Management
* Donated Management
* Backers Management
* Multi Currencies
* 3 Payment Gateways ( Authorize.net, Paypal, Stripe )

== Installation ==

1. Upload the plugin files to the `/wp-content/plugins/tp-donation` directory, or install the plugin through the WordPress plugins screen directly.
1. Activate the plugin through the 'Plugins' screen in WordPress
